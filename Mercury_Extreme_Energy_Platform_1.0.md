# NASA-AOD 專案概念建議書：Mercury Extreme Energy Platform (MEEP) 1.0

**文件編號：ILPA-MC-2026-001**  
**計畫階段：Phase-A 概念架構與系統可行性論證**  
**核心目標：於水星南極永晝區部署大規模太陽能平台，為深空探測任務及太陽系內能源中繼提供穩定電力。**

## 一、專案目的

### 1.1 核心目標

> 利用水星極強嘅日照（地球 6.6 倍），建立一個 **1 km²** 級別嘅太陽能發電平台，為以下任務提供能源：
> - 水星科學基地
> - 深空探測器（前往外太陽系）
> - 太陽系內能源中繼站

### 1.2 水星嘅獨特優勢

| 參數 | 數值 | 對比地球 |
|:---|:---|:---|
| 日照強度 | ~9,000 W/m² | **6.6 倍** |
| 極區日照時數 | ~7,000 小時/年 | ~1.6 倍 |
| 日夜溫差 | ~600°C | 最大溫差資源 |
| 重力 | 0.38g | 發射/建設更容易 |

## 二、與 NASA 及國際任務嘅配合

### 2.1 對齊現有及未來任務

| 任務 | 時間 | 配合方式 |
|:---|:---|:---|
| **BepiColombo**（ESA/JAXA） | 2026 入軌 | 利用其數據優化選址 |
| **MExAG 水星著陸器建議** | 未來 | 為著陸器提供能源 |
| **太陽帆任務概念**（布朗大學） | 概念階段 | 太陽帆運輸 + 本平台供電 |
| **Discovery Program** | 未來十年 | 定位為 Discovery 級能源方案 |

### 2.2 戰略定位

| 層級 | 任務 | 本專案角色 |
|:---|:---|:---|
| 內太陽系 | 月球（地球供電） | 互補 |
| 內太陽系 | 金星（科學平台） | 互補 |
| **內太陽系邊緣** | **水星（深空中繼）** | **能源前哨** |

**本專案係人類前往外太陽系嘅能源跳板。**

## 三、核心技術

### 3.1 耐高溫太陽能系統

| 技術 | 說明 |
|:---|:---|
| 太陽能板 | 耐高溫 GaAs 多接面（可承受 450°C） |
| 效率 | 40%（水星環境下） |
| 冷卻 | 高反射塗層 + 被動散熱 + 備用液體冷卻 |
| 部署位置 | 水星南極永晝區（避免極端高溫） |

### 3.2 熱差發電（TEG）

| 參數 | 數值 |
|:---|:---|
| 溫差 | ~600°C（日夜） |
| 熱電材料 | 方鈷礦（Skutterudite） |
| 效率 | ~12% |
| 年貢獻 | ~3.0 × 10¹⁶ J（約 830 億度電） |

### 3.3 雷射能量傳輸

| 參數 | 數值 |
|:---|:---|
| 傳輸方式 | 1064 nm 雷射 + 自適應光學（AO） |
| 效率 | 85% |
| 接收站 | 地球高海拔沙漠 / 軌道中繼衛星 |

### 3.4 儲能系統

| 技術 | 功能 |
|:---|:---|
| 飛輪儲能 | 平滑功率輸出 |
| 超級電容 | 應對突發峰值需求 |

## 四、能量產出與成本

### 4.1 能量產出

| 參數 | 數值 |
|:---|:---|
| 平台面積 | 1 km² |
| 年發電量（月面） | ~2.8 × 10¹⁷ J |
| 雷射傳輸效率 | 85% |
| **地球接收能量** | **~2.4 × 10¹⁷ J（約 6,700 億度電）** |
| 可供應家庭 | ~1.9 億戶 |

### 4.2 成本拆解

| 項目 | 成本（億美元） |
|:---|:---|
| 發射（13 次 Starship） | 19.5 |
| 太陽能板（1 km²） | 2.0 |
| 耐高溫結構 | 1.0 |
| 熱差發電模組 | 0.5 |
| 雷射傳輸系統 | 1.0 |
| 儲能與控制 | 0.5 |
| 研發與測試 | 0.5 |
| **總計** | **25.0** |

### 4.3 投資回報

| 參數 | 數值 |
|:---|:---|
| 年收入（工業電價 $0.05/度） | 335 億美元 |
| 回本期 | **約 27 天（理論）/ 1-2 年（現實）** |
