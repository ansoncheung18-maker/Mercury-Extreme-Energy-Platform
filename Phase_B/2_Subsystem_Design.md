# 水星專案 Phase B：子系統設計（Subsystem Design）

**版本：1.0**  
**更新日期：2026-05-30**

---

## 1. 太陽能子系統

### 1.1 設計目標
- 提供 2.92 GW 峰值功率
- 抵抗 430°C 極端高溫
- 抵抗高輻射環境

### 1.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 電池類型 | 耐高溫 GaAs 多接面 | 可承受 450°C |
| 安裝方式 | 水平鋪設 | |
| 冷卻方式 | 液態鈉主動冷卻 | 保持 150°C 以下 |
| 防護 | 高反射塗層 | 減少熱吸收 |

### 1.3 質量與功率

| 參數 | 數值 |
|:---|:---|
| 總面積 | 1 km² |
| 面密度 | 0.3 kg/m² |
| 總質量 | 300 噸 |
| 效率 | 40% |
| 總發電量 | 2.92 GW |
